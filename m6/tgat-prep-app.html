TGAT ติวเตอร์เตรียมสอบ

ฝึกทำข้อสอบ TGAT แบบจับเวลาจริง

เลือกพาร์ทที่ต้องการฝึก ทำข้อสอบในเวลาที่กำหนด แล้วดูผลคะแนน เฉลย และจุดที่ต้องพัฒนา (สุ่มลำดับข้อและตัวเลือกให้อัตโนมัติทุกครั้ง)

เลือกพาร์ทที่ต้องการฝึก (เลือกได้มากกว่า 1)
นำเข้าข้อสอบเอง (ไม่ต้องแก้โค้ด)

อัปโหลดไฟล์ .csv หรือ .json เพื่อใช้ข้อสอบของคุณเองแทนชุดตัวอย่าง — แนะนำให้กด ดาวน์โหลดเทมเพลต ไปแก้ใน Excel / Google Sheets ก่อน แล้วค่อยอัปโหลดกลับ

📖 วิธีเตรียมไฟล์ข้อสอบ
แบบที่ 1 : ไฟล์ CSV (แก้ใน Excel / Google Sheets — แนะนำ)

แต่ละแถว = ข้อสอบ 1 ข้อ ใช้หัวคอลัมน์ดังนี้ (รองรับทั้งข้อสอบ 4 และ 5 ตัวเลือก — เว้นคอลัมน์ "จ" ว่างได้ถ้ามีแค่ 4 ตัวเลือก)

คอลัมน์ความหมาย
ส่วนชื่อส่วน เช่น "ส่วนที่ 1: ทักษะการพูด (Speaking)"
ประเภทประเภทข้อสอบ เช่น "การถาม–ตอบ"
รหัสบทความเว้นว่าง = ข้อเดี่ยว / ใส่รหัสเดียวกัน เช่น A1 = ข้อที่ใช้บทความ/บทสนทนาเดียวกัน
บทความเนื้อบทความ/บทสนทนา (ใส่เฉพาะแถวแรกของชุดที่รหัสเดียวกันก็พอ)
คำถามโจทย์
ก ข ค ง จตัวเลือก 4-5 คอลัมน์ ("จ" เว้นว่างได้ถ้ามี 4 ตัวเลือก)
คำตอบข้อที่ถูก ใส่ได้ทั้ง ก/ข/ค/ง/จ, A/B/C/D/E หรือ 1-5
เฉลยคำอธิบายเฉลย (จะเว้นว่างก็ได้)

คะแนนจะถ่วงน้ำหนักให้แต่ละ "ส่วน" เท่ากันอัตโนมัติให้รวมเป็น 100

แบบที่ 2 : ไฟล์ JSON (กำหนดคะแนน/เวลาเองได้) 
{
  "name": "ชื่อชุดข้อสอบ",
  "minutes": 60,
  "sections": [
    { "name": "ชื่อส่วน", "score": 100,
      "groups": [
        { "name": "ชื่อประเภท",
          "questions": [
            { "q": "คำถาม...", "choices": ["ก","ข","ค","ง","จ"],
              "answer": 0, "explain": "คำอธิบาย" }
          ] },
        { "name": "ใช้บทความร่วมกัน",
          "sets": [
            { "passage": "บทความ...",
              "items": [ { "q":"...", "choices":["..",".."],
                          "answer":0, "explain":"" } ] }
          ] }
      ] }
  ]
}

answer นับเริ่มจาก 0 (0=ก, 1=ข, 2=ค, 3=ง, 4=จ) — ใส่ choices กี่ตัวเลือกก็ได้
